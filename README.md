# Canada28 模拟器 (Canada28 Simulator)

**专业级加拿大28彩票数据分析与模拟工具**

一款集成了实时数据同步、历史回测、负追策略模拟（D'Alembert）、冷热号分析以及完整授权管理系统的桌面应用程序。

## ✨ 主要功能

*   **实时模拟**:
    *   自动同步最新的开奖数据（日均402期）。
    *   实时计算盈亏，可视化展示资金曲线。
    *   支持挂机模式，自动根据策略下注。

*   **智能策略 (负追/D'Alembert)**:
    *   内置 "亏损加注，盈利减注" 的 D'Alembert 变体策略。
    *   **亏损时**: 自动增加注码，以此快速回本。
    *   **盈利时**: 自动减少注码，保护利润。
    *   可自定义调整起步注额、加注步长、减注频率等参数。

*   **数据分析**:
    *   **冷热号码表**: 实时统计最热和最冷的号码。
    *   **自定义冷门导出**: 支持设定 "过去N期" + "出现率<X%" 的冷门筛选，并支持导出纯数字格式。
    *   **热门组合**: 分析出现频率最高的号码组合。
    *   **全局搜索**: 在历史数据库中搜索特定号码。

*   **企业级授权系统**:
    *   **机器码绑定**: 一机一码，防止盗版。
    *   **网络时间校验**: 强制联网验证，防止修改本地时间作弊。
    *   **有效期控制**: 支持按天数（月卡/季卡/年卡）授权。

## 🛠️ 安装与运行

### 1. 运行环境
*   Windows 10/11 (x64/x86)
*   需要联网 (启动时必须校验授权)

### 2. 开发者运行
```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 启动模拟器
python Canada28/canada28_simulator_qt.py

# 3. 启动注册机 (仅管理员)
python Canada28/keygen.py
```

### 3. 打包发布
使用内置脚本一键打包：
```bash
python Canada28/build_exe.py
```
这将在 `dist` 目录下生成：
*   `Canada28Simulator_Client.zip` (发给客户)
*   `KeyGen_Admin_Tool.zip` (管理员自用)
